Introducing EduSpike - A First Step in Reimagining Neural Simulation Tools

Eduspike is an educational neuronal simulation tool that provides a modernised interface for interacting with computational neuron models. Through a powerful visualisation module, built-in IDE and accompanying python library, we provide users a hands on experience with popular models such as the LIF and the Hodgkin Huxley neurons. Modelling computational neurons has never been more intuitive.
